# Comprehensive Stock & E-Commerce Management System 

A robust, multi-role monolithic web application built with **Laravel**, designed to seamlessly handle inventory tracking, order processing, and online sales.

##  Key Features
- **Multi-Role Access Control:** Custom middleware protecting routes for Admins, Managers, Staff, and Customers.
- **Secure Checkout:** Integrated with **Stripe Payment Gateway** for safe and seamless transactions.
- **Advanced Inventory:** Dynamic QR Code generation for physical product tracking and management.
- **Event-Driven Architecture:** Utilizes Laravel Events & Listeners to automatically handle user state changes and notifications.
- **Customer Interactions:** Features a complete product review/rating system and automated email notifications (Mailables).

##  Tech Stack
- **Backend:** PHP, Laravel Framework
- **Frontend:** Blade Templates, Bootstrap/Tailwind (Vite)
- **Database:** MySQL
- **Integrations:** Stripe API, QR Code Generator
